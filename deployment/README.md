# Deployment and Operations From Scratch

Status: **Planned**

This track will study how software moves from source code to a running,
observable, and recoverable production service.

## First-Principles Questions

- What must exist for a program to run outside a developer's machine?
- How can builds be repeatable?
- How are configuration and secrets supplied safely?
- How do we detect and recover from a bad release?
- How do we know whether a running service is healthy?

## Planned Outcomes

- Build repeatable application artifacts
- Understand containers and runtime environments
- Create continuous integration and delivery pipelines
- Manage configuration, secrets, migrations, and releases
- Use logs, metrics, traces, alerts, and health checks
- Practice rollbacks, backups, recovery, and incident response

Lessons and exercises will be added after the prerequisite tracks are complete.
