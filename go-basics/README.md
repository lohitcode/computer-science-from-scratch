| Track | Description | For |

| **DSA** | LeetCode-style problems (like c-dsa) | Practice & interviews |

---

## 🎯 Why Go?

**For your goals:**
- ✅ Performance → Faster than Python
- ✅ Microservices → Go excels at this
- ✅ Cloud native → Kubernetes, Docker written in Go
- ✅ Backend jobs → Gin/Echo frameworks (like FastAPI)
- ✅ Static typing → Catch errors early

## 🚀 Quick Start

```bash
cd go-basics
go run main.go
```

**Or build binary:**
```bash
go build main.go
./main
```
