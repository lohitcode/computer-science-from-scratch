# Computer Networking From Scratch

Status: **Planned**

This track will study how computers communicate, from electrical and packet
delivery concepts through the protocols used by backend services.

## First-Principles Questions

- How does information travel between two machines?
- Why is networking divided into layers?
- How are machines and processes addressed?
- What happens between entering a URL and receiving a response?
- How do networks detect, recover from, and tolerate failure?

## Planned Outcomes

- Understand packets, frames, addresses, routing, and ports
- Explain the roles of Ethernet, IP, TCP, UDP, DNS, TLS, and HTTP
- Use common network inspection and debugging tools
- Build simple TCP and HTTP clients and servers
- Diagnose latency, connection, name-resolution, and protocol problems
- Design networked applications with failure in mind

Lessons and exercises will be added after the prerequisite tracks are complete.
