# Production Software Projects

Status: **Planned**

This track will combine the repository's programming, DSA, systems, database,
networking, security, and deployment lessons into complete software.

## First-Principles Questions

- What real user problem does the project solve?
- Which requirements are essential for the first usable version?
- How will correctness, performance, and reliability be verified?
- How will another engineer understand and maintain the system?
- What evidence shows that the software is production-ready?

## Planned Outcomes

- Turn ambiguous requirements into an executable project plan
- Build and document complete backend or systems applications
- Use tests, reviews, automation, and version control effectively
- Integrate databases, networks, security, and observability
- Deploy software and operate it under realistic failure conditions
- Produce portfolio-quality projects with clear engineering decisions

Projects will be selected after the prerequisite tracks are complete.
