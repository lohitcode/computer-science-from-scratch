## 🐍 Python vs JavaScript Quick Reference

| Concept | JavaScript | Python |
|---------|-----------|---------|
| Variable | `let x = 1` | `x = 1` |
| Function | `function f() {}` | `def f():` |
| Async | `async function()` | `async def` |
| Array | `[1, 2, 3]` | `[1, 2, 3]` (list) |
| Object | `{key: val}` | `{"key": "val"}` (dict) |
| Null | `null` | `None` |
| Try/catch | `try {} catch {}` | `try: except:` |
| Boolean | `true/false` | `True/False` |
| And/Or | `&& / \|\|` | `and / or` |
| Not | `!` | `not` |
